question1.py代码运行所需环境：
本代码读取和输出csv文件所用路径都是相对路径。在本py文件所在目录放置question1代码处理好的车辆数据文件夹（每辆车一个文件夹，用车辆ID命名，如‘AA00002’）
代码运行结束后会在上述车辆数据文件夹里生成对应路段的图片。

question1.py代码内容说明：
question1代码前140行主要是用来定义急加速急减速识别算法、熄火滑行识别算法、平均速度计算算法，以及获取本代码所在目录路径和车辆数据文件夹的信息。
从140行开始是主函数，用来读取车辆数据文件，调用前140行定义的算法来计算急加速次数、平均速度等等信息，然后用这些信息以及车辆数据中的经纬度信息来画题目一要求的经纬度坐标系图。


draw.py代码说明：
draw.py调用folium包，利用车辆数据的经纬度信息来画出html格式的具有地图背景的行车线路图。
每次只能画一辆车的所有路段的html图，使用时需修改第六、七行的路径为本地车辆数据文件夹的路径，并修改最后一行代码中的数字为本车辆的路段数。运行后会在车辆数据文件夹中生成html图，用浏览器打开即可。
